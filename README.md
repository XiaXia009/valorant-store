# Valorant Store
Python script to display Valorant skin IDs in your store


## Usage
```python
from valorant import Valorant

vlr = Valorant("username", "password", "eu")
print(vlr.store())
```


## Get More Information by ID
https://valorant-api.com/v1/weapons/skinlevels/{uuid}
